# WeatherDataAnalyzer
This project is a Java-based weather data analyzer that processes CSV files, computes statistics, and generates structured reports. It supports temperature conversion, filtering by month, precipitation analysis, and categorization of weather conditions.
Features
* Load and process weather data from a CSV file
* Convert temperatures between Celsius and Fahrenheit
* Compute average temperature for a given month
* Count the number of rainy days
* Identify days exceeding a temperature threshold
* Categorize weather conditions (e.g., Hot, Warm, Cold)
* Generate a structured report for easy analysis
